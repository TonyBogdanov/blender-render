Quick-and-easy SheepIT renderer for Ubuntu VM setup.

# Install

```bash
sudo su
```

```bash
apt-get update && \
apt-get upgrade -y && \
apt-get install git && \
git clone https://github.com/TonyBogdanov/blender-render.git ~/blender-render && \
cd ~/blender-render && \
./setup.sh

```
